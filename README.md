بوت هاي داي - نظام أتمتة زراعة وحصاد البيع

نظرة عامة

تطبيق أتمتة شامل لـ Hay Day يعمل على محاكي BlueStacks، يقوم بـ
الزراعة التلقائية للمحاصيل
الحصاد الذكي عند نضج المحصول
البيع الآلي في المتجر
تتبع الأرباح والإحصائيات

المتطلبات

Python 3.8 أو أحدث
BlueStacks محاكي (تشغيل)
لعبة Hay Day (مثبتة وتعمل في BlueStacks)

التثبيت

1. استنسخ المستودع
git clone https://github.com/nutrweqxfjo-ctrl/hay-day-bot.git
cd hay-day-bot

2. ثبت المكتبات
pip install -r requirements.txt

3. إعداد الإعدادات (اختياري)
عدّل ملف config.py حسب احتياجاتك

الاستخدام

البدء السريع

python main.py

الخطوات قبل البدء

1. افتح BlueStacks وتأكد من أنه يعمل
2. شغّل لعبة Hay Day داخل BlueStacks
3. ضع اللعبة في الوضع الأفقي (Landscape)
4. قف على الشاشة الرئيسية للعبة
5. شغّل البوت من خلال تنفيذ الأمر أعلاه

المميزات الرئيسية

1. كشف الحالة الذكي
تحليل شاشة اللعبة في الوقت الفعلي
كشف المحاصيل الجاهزة للحصاد
تحديد الحقول الفارغة
مراقبة المخزون

2. إدارة المحاصيل
زراعة محاصيل متعددة
تتبع وقت النمو
حصاد ذكي عند النضج
اختيار المحاصيل الأكثر ربحية

3. نظام البيع
بيع تلقائي في المتجر
حساب الأرباح الديناميكية
تتبع سجل المبيعات
تحسين الأسعار

4. التسجيل والإحصائيات
سجل مفصل لجميع العمليات
إحصائيات يومية
تتبع الأرباح الإجمالية
تقارير الأداء

هيكل الملفات

hay-day-bot/
main.py                 البرنامج الرئيسي

README.md             هذا الملف

الإعدادات المتقدمة

تعديل سرعة الدورة
في config.py:
CYCLE_INTERVAL = 2

تغيير المحاصيل المفضلة
CROPS_CONFIG = {
    'wheat': {
        'growth_time': 120,
        'profit': 50
    },
}

تعديل سلوك البيع
MIN_ITEMS_TO_SELL = 10
SELL_PRICE_MULTIPLIER = 1.2

استكشاف الأخطاء

المشكلة: البوت لا يكتشف اللعبة
الحل: تأكد من أن BlueStacks مفتوح وأن نافذته نشطة
تحقق من دقة الشاشة (يجب أن تكون 1280x720)

المشكلة: البوت لا يضغط بشكل صحيح
الحل: قد تحتاج إلى معايرة مواضع الأزرار في config.py

المشكلة: كشف المحاصيل غير دقيق
الحل: عدّل نطاقات الألوان في CROPS_CONFIG

السجلات

سيتم حفظ جميع السجلات في ملف hayday_bot.log

عرض السجلات الحية
tail -f hayday_bot.log

عرض آخر 100 سطر
tail -n 100 hayday_bot.log
